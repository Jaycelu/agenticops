# AgenticOps README v1 设计说明

日期：2026-03-10

## 目标

为仓库生成首个正式版本 README，兼顾两类读者：

- 想快速理解产品价值和界面能力的业务读者
- 想立即启动项目和查看结构的开发者

## 方案选择

采用“平衡型 README”：

- 顶部先说明项目定位和主链路
- 中段展示核心能力、预览图和典型使用链路
- 后段提供快速启动、配置说明、项目结构和路线图

不采用纯营销型或纯工程型结构，避免首版开源文档失衡。

## 设计原则

- 中文为主，顶部保留英文 README 入口
- 使用标准开源 README 结构：简介、特性、预览、快速开始、结构、路线图
- 以当前代码和实际页面为准，不沿用历史 Streamlit 文案
- 首屏只保留一张总览图，避免图片过载
- 文案强调 AgenticOps 主链路，而不是单页功能罗列

## 截图编排策略

README 中的演示图建议采用两层结构：

1. 首屏总览图
2. 核心页面组图

推荐顺序：

1. 驾驶舱
2. 事件中心
3. Case 中心
4. 执行中心
5. 智能体中心
6. 记忆中心
7. 日志中心
8. Zabbix 中心
9. 资产拓扑

## 配置策略

首版 README 只展示必要配置项，不直接暴露任何真实地址或凭据。

同时修正 `deploy/env.example` 中的示例值，改为安全占位符，避免把真实配置作为公开仓库样例发布。
