# AgenticOps v0.1.0

首个公开版本。

## 亮点

- 建立 `Event -> Case -> Multi-Agent -> Memory -> Fabric` 主链路
- 提供中文主 README 与英文入口 README
- 补齐快速启动、部署说明、端口说明与发布前检查清单
- 清理公开仓库中的日志、采集结果、本地会话历史和私有部署痕迹
- 将示例配置、脚本与服务模板改为公开可复用版本
- 添加 `MIT` 许可证

## 包含的核心模块

- 驾驶舱
- 事件中心
- Case 中心
- 执行中心
- 智能体中心
- 记忆中心
- 日志中心
- Zabbix 中心
- 资产拓扑
- 工单
- 设置

## 技术栈

- 前端：Vue 3 + Vite + TypeScript
- 后端：FastAPI + SQLAlchemy
- 数据库：PostgreSQL

## 说明

- 当前版本更适合作为公开首版和后续演进基线
- 如需生产使用，仍建议根据实际环境补齐部署、监控、权限和截图脱敏流程
